# RollMarket
RollMarket

                                                Project is Using these -> 
                                                
                                                         
1 : jetPack Compose 
2 : Navigation component
3 : Room and Local Database
4 : retrofit with kotlin Coroutines
5 : liveData
6 : mvvm Architecture
7 : Koin (dependency injection )
8 : Coil
9 : Lottie Animation
